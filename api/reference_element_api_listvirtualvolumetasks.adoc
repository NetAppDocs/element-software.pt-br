= ListVirtualVolumeTasks
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `ListVirtualVolumeTasks` método para obter uma lista de tarefas de volume virtual no sistema.



== Parâmetros

Este método tem o seguinte parâmetro de entrada:

|===
| Nome | Descrição | Tipo | Valor padrão | Obrigatório 


 a| 
VirtualVolumeTaskIDs
 a| 
Uma lista de IDs de tarefas de volume virtual para os quais recuperar informações. Se você omitir esse parâmetro, o método retornará informações sobre todas as tarefas de volume virtual.
 a| 
Array UUUID
 a| 
Nenhum
 a| 
Não

|===


== Valor de retorno

Este método tem o seguinte valor de retorno:

|===


| Nome | Descrição | Tipo 


 a| 
tarefas
 a| 
Uma lista de objetos que descrevem as tarefas de volume virtual no cluster.
 a| 
xref:reference_element_api_task_virtual_volumes.adoc[tarefa] array

|===


== Exemplo de solicitação

As solicitações para este método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "ListVirtualVolumeTasks",
    "params": {
    },
    "id": 1
}
----
