= ListSchedules
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar `ListSchedules` para obter informações sobre todos os snapshots programados que foram criados.



== Parâmetros

Este método não tem parâmetros de entrada.



== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
programações
 a| 
Uma lista das programações atualmente no cluster.
 a| 
xref:reference_element_api_schedule.adoc[programação] array

|===


== Exemplo de solicitação

As solicitações para este método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "ListSchedules",
    "params": {},
"id": 1
}
----
