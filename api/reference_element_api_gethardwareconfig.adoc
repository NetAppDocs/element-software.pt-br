= GetHardwareConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `GetHardwareConfig` método para obter as informações de configuração de hardware para um nó. Estes dados de configuração destinam-se a utilização interna. Para obter um inventário de componentes de hardware de sistema ativo mais útil, use o `GetHardwareInfo` método.



== Parâmetros

Este método não tem parâmetros de entrada.



== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


| HardwareConfig | Lista de informações de hardware e configurações atuais. | Objeto JSON 
|===


== Exemplo de solicitação

As solicitações para este método são semelhantes ao seguinte exemplo:

[listing]
----
{
   "method": "GetHardwareConfig",
   "params": {},
   "id" : 1
}
----
